# Portis demo use case

## Install

With package.json, install demo locally with:  

- Make link to newly custom `portiswrp` package: `npm install --save ../portiswrp`
- Install dependencies: `npm install`

### Build & Run

- Build: `npm run build`
- Run: `npm run start`

## Use portiswrp

`const portis = require('portiswrp');`
