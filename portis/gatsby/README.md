# Documentation

## Prerequisite

Install all dependencies:

- `npm install @portis/web3`
- `npm install web3`

## Build and Run

- Build and Run (dev): `npm run start`
- Build: `npm run build`
- Run (prod): `npm run serve`

In dev mode, page is server on `localhost:8000`
